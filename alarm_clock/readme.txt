# Alarm Clock 🔔

A command-line alarm clock built with Python that plays an audio 
alert when the set time is reached.

## Features
- Set alarm in HH:MM:SS format
- Plays audio when alarm triggers
- Displays current time while waiting
- Input validation with error handling

## Requirements
- Python 3.x
- pygame

## Installation
pip install pygame

## How to Use
1. Run the script: python main.py
2. Enter desired alarm time in HH:MM:SS format (e.g. 14:30:00)
3. Wait for alarm to trigger
4. Audio plays when alarm time is reached

## Technologies Used
- datetime module
- pygame
- os module
