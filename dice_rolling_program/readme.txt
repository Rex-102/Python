# Dice Roller 🎲

A command-line dice roller that displays ASCII art dice and 
calculates the total value of all rolled dice.

## Features
- Roll any number of dice simultaneously
- Visual ASCII art dice display
- Automatic total calculation
- Random dice generation

## Requirements
- Python 3.x
- No external dependencies

## How to Use
1. Run the script: python main.py
2. Enter the number of dice to roll
3. View your dice results and total
